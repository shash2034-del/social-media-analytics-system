# Social Media Analytics System

## 📌 Project Overview
This project analyzes social media data using PostgreSQL and AI.

## 🚀 Features
- User, Post, Like, Comment management
- SQL analytics queries
- Sentiment Analysis using TextBlob
- Streamlit dashboard

## 🛠 Tech Stack
- PostgreSQL
- Python
- Streamlit
- TextBlob

## ▶️ Run Project
```bash
pip install -r requirements.txt
streamlit run app.py
